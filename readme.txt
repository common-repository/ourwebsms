== Installation ==

This section describes how to install the plugin and get it working.


1. Upload `ourwebsms` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to ourwebsms under setting menu and  fill out the form
4. Use [ourwebsms] shortcode or use the ourwebsms wiget
5. Once you receive your first signup, ourwebsms will send you a sms message to complete the account setup process
6. Log on http://ourwebsms.com to start sending sms's
